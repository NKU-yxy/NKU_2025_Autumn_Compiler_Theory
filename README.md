## 平时分数

编译实验&答辩（35%） + 书面作业（15%）

鉴于编译原理几乎是本科期间最难的一门课，实验（含答辩）自然也是十分折磨人且困难的。

老师和助教都是好人，只是这个实验就是很难且无法避免。

唯一需要注意的就是，我们这届代码优化具有额外加分，所以如果下届还有可以考虑多拿一点额外加分&&书面作业要认真做，不然可能会有很意想不到的扣分。

本人实验拿了32/35 书面拿了13/15 加起来45，应该算已经很高的了。

## 期末考试

这个占50% wg老师也非常好，会在考试前画重点+发前两年的卷子和答案，但十分不建议考试前一两天开始突击，如果只想及格可以看看词法分析大题+语法分析第一问+选择题好好学，如果追求绩点本人建议至少提前3-5天开始复习。不然实在不太来得及。
